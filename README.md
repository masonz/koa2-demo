# koa2-demo
简单的Koa2案例

### Technology

  - koa-middlewares
    - koa-router
    - koa-multer
    - koa-logger
    - koa-hetml
    - koa-jwt
    - koa-bodyparser
    - koa-static2
    - koa-views
  - mongodb & mongoose
  - jsonwebtoken(JWT)
  - aglio/api-blueprint
  - babel

### Installation

```javascript 

  $ npm install // install
  
  $ npm start // run
  
  open url localhost:3001

```
